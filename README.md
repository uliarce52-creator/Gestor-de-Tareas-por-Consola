# Gestor de Tareas en Consola (CLI Task Manager)

Este es un proyecto práctico desarrollado en Python para consolidar mis conocimientos en los fundamentos del lenguaje. El objetivo principal fue aplicar lógica de programación básica sin dependencias externas.

## 🚀 Conceptos Aplicados
* **Ciclos (`while`, `for`)**: Control del flujo del programa y recorrido de la lista de tareas.
* **Condicionales (`if`, `elif`, `else`)**: Toma de decisiones según la opción elegida por el usuario.
* **Estructuras de Datos (`listas`)**: Almacenamiento y manipulación de las tareas en memoria.

## 🛠️ Cómo Ejecutar el Proyecto

1. Asegúrate de tener Python instalado en tu computadora.
2. Descarga o clona este repositorio.
3. Abre la terminal en la carpeta del proyecto y ejecuta:
   ```bash
   python MAIN.py
   ```
